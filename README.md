# 🌟 Predict Stellar Class

A Machine Learning project developed for the Kaggle competition **Playground Series - Season 6, Episode 6**. The objective is to predict the stellar class of celestial objects using observational and physical features.

## 🎯 Project Goal

The goal of this project is to build a classification model capable of accurately identifying the stellar class of astronomical objects based on the provided dataset.

The project includes:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Model training
* Model evaluation
* Kaggle submission generation

## 📂 Project Structure

```text
Predict_Steller_Class/
│
├── train.csv
├── test.csv
├── sample_submission.csv
├── notebook.ipynb
├── submission.csv
├── requirements.txt
└── README.md
```

### File Description

* **train.csv** – Training dataset used for model development.
* **test.csv** – Dataset used to generate predictions.
* **sample_submission.csv** – Kaggle submission template.
* **notebook.ipynb** – Complete workflow including preprocessing, training, and evaluation.
* **submission.csv** – Final prediction file.
* **requirements.txt** – Required Python libraries.
* **README.md** – Project documentation.

```bash

```

Install required packages:

